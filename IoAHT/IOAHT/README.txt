The project is made for the health monitoring of Livestocks.
project_server.py file is used to run server for handling MYSQL database.
Project also uses thingSpeaks cloud for better data visualizations and con be monitored using nad third-party android application for ThingSpeak.
This project includes various prebuilt libraries for handling sensor data efficiently like DHT11 and BMP180 Sensor which can be found on internet.
Connect the board, go through the source code and accordingly connect sensors, run server(create your own server), make changes according to the server and table names in .py file
